# 接口文档（REST API 草案）

## 认证与用户（待实现）
- `POST /api/auth/register`：注册 `{username, email, password}` → `{id, username}`。
- `POST /api/auth/login`：登录 `{username, password}` → `{token, user}`。
- `GET /api/users/me`：获取当前用户信息（需 `Authorization: Bearer`）。

## 算法目录（待实现）
- `GET /api/algorithms`：获取算法列表。
- `GET /api/algorithms/{id}`：算法详情。
- `POST /api/algorithms`：新增（管理员）。
- `PUT /api/algorithms/{id}`：更新（管理员）。
- `DELETE /api/algorithms/{id}`：删除（管理员）。

## 运行记录（待实现）
- `POST /api/runs`：保存运行 `{algorithmId, inputJson, paramsJson, stepCount, durationMs}` → `{id}`。
- `GET /api/runs/{id}`：运行详情。
- `GET /api/runs?userId=...&algorithmId=...`：筛选查询。

## 收藏（待实现）
- `POST /api/favorites`：收藏算法 `{algorithmId}` → `{id}`。
- `GET /api/favorites`：当前用户收藏列表。
- `DELETE /api/favorites/{id}`：取消收藏。

## 统计（待实现）
- `GET /api/stats/overview`：用户数、算法数、运行数、热门算法。

## 健康检查（已实现）
- `GET /api/health`
  - 响应示例：
    ```json
    {"status":"UP","service":"algorithm-visualizer-backend"}
    ```

## 错误约定
- 统一响应格式：`{ code, message, data? }`（待实现）。
- 常见错误码：`400 参数错误`、`401 未授权`、`403 禁止访问`、`404 未找到`、`500 服务器异常`。
